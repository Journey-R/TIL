# Github를 통한 웹 서버 운영

1. 웹 서버 운영을 위한 파일들을 repository에 올림
2. 해당 repository의 settings 페이지로 이동
3. GitHub Pages  - Source에서 branch 선택 후 Save
   - None -> master branch
4. Your site is ready to be published at : 주소
